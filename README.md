# System-programming
Лекция от 02.02.2026
 Нужно создать  восемнадцатиричную таблицу умножения

 Таблица должна состоять из 10 цифр и букв A-11,B-12,C-13,D-14,E-15,F-16,G-17,H-18

 После пересечения нашего лимита то  впереди ставим количество пересечения.

Например: в десятичной системе число 15. единица это количество пересечений. пятерка это число из системы. 

Для начала надо просто понять как сделать простую таблицу умножение в c++. 

#include <iostream>

		for (int j = 1; j <= 10; j++) {

		}

		cout << endl; 


 }


}



 Директивы препроцессора #include
#include <iostream> — подключает стандартную библиотеку ввода-вывода C++, предоставляя объекты cin, cout, cerr для работы с потоками данных.
#include <iomanip> — подключает манипуляторы форматирования потоков, в частности setw(), используемый для управления шириной поля вывода.

Два вложенных цикла:
Внешний цикл (i) перебирает строки от 1 до 10.
Внутренний цикл (j) перебирает столбцы от 1 до 10.
Для каждой пары (i, j) вычисляется произведение i * j.

Форматирование вывода:
setw(2) — выделяет под каждое число ровно 2 символа, чтобы таблица выглядела аккуратно.
После каждого числа добавляется пробел для разделения.

Первый этап завершён. Осталось понять как результат умножение перевести в нужную  мне систему.

Обработка нуля: если число равно 0, результат — "0".
Обработка отрицательных чисел: запомнить знак, работать с модулем числа.
Цикл деления:
Делим число на 18.
Остаток от деления определяет текущую цифру:
0–9 → символы '0'–'9'
10–17 → символы 'A'–'H'
Добавляем цифру в строку (в обратном порядке).
Повторяем, пока число не станет равным 0.
Разворот строки: так как цифры собираются от младших разрядов к старшим, строку нужно развернуть.
Добавление знака: если исходное число было отрицательным, добавляем '-' в начало


#include <iostream>
#include <string>
#include <algorithm>
#include <iomanip>

using namespace std;

// Функция для перевода числа из десятичной в 18-ричную систему счисления
string decimalToBase18(int decimal) {
    if (decimal == 0) return "0";

    const string digits = "0123456789ABCDEFGH"; // Символы для 18-ричной системы
    string result = "";

    while (decimal > 0) {
        int remainder = decimal % 18;
        result += digits[remainder];
        decimal /= 18;
    }

    reverse(result.begin(), result.end());
    return result;
}

int main() {
    for (int i = 1; i <= 10; i++) {
        for (int j = 1; j <= 10; j++) {
            cout << setw(2) << decimalToBase18(i * j) << " ";
        }
        cout << endl;
    }
    return 0;
}
